# ComicTeach launch checklist

Five tasks, ~15 minutes total. Do them in order. Each one unlocks the next.

---

## 1. Create the GitHub organization (3 min)

We're putting both the marketing site and the agent under a shared GitHub
organization so the repos look like a real company (and so you can add
co-maintainers later without changing URLs).

1. Sign in to GitHub as `xw2871` (or your preferred handle).
2. Open <https://github.com/account/organizations/new>.
3. Pick the **Free** plan.
4. Organization name: **`comicteach`** (lowercase, no dashes).
5. Contact email: your NYU email.
6. Skip "invite members" — you can add them later.

Verify the org exists at <https://github.com/comicteach>.

---

## 2. Create the two repos (4 min)

### Repo A — the agent (open-source product)

1. <https://github.com/organizations/comicteach/repositories/new>
2. Name: **`comicteach`**
3. Description: `Open-source AI agent that turns lessons into manga.`
4. Visibility: **Public**
5. Do **not** initialize with README, .gitignore, or license (we'll push our own).
6. Click **Create repository**.

Push your existing agent code:

```bash
cd ~/ai-comic-book                     # the standalone agent folder
git init -b main
git add .
git commit -m "Initial commit"
git remote add origin git@github.com:comicteach/comicteach.git
git push -u origin main
```

### Repo B — the website (this project)

1. <https://github.com/organizations/comicteach/repositories/new>
2. Name: **`comicteach-site`**
3. Description: `Marketing & community site for ComicTeach.`
4. Visibility: **Public**
5. Do **not** initialize with anything.
6. Click **Create repository**.

Push the site:

```bash
cd ~/comicteach-site                   # after running setup_standalone.sh
git init -b main
git add .
git commit -m "Initial commit"
git remote add origin git@github.com:comicteach/comicteach-site.git
git push -u origin main
```

## 3. Create the Discord server (5 min)

1. Open Discord (desktop or web).
2. Click the **+** in the left rail → **Create My Own** → **For me and my friends**.
3. Server name: **`ComicTeach`**.
4. Upload a server icon (drag in `public/favicon.svg` or your own image).
5. Hit **Create**.

Add channels for the community vibe. Right-click the **TEXT CHANNELS** category → **Create Channel**:

| Channel              | Purpose                                              |
| -------------------- | ---------------------------------------------------- |
| `#welcome`           | New members say hi; pinned getting-started message   |
| `#announcements`     | Releases + blog posts (read-only for non-mods)       |
| `#show-and-tell`     | Comics generated by users                            |
| `#feature-requests`  | What ComicTeach should do next                       |
| `#help`              | Setup + troubleshooting                              |
| `#dev`               | Repo discussion, PRs, design                         |

Create the **permanent invite link**:

1. Right-click the server name (top-left) → **Invite People**.
2. Click **Edit invite link**.
3. Expire after: **Never**.
4. Max uses: **No limit**.
5. Click **Generate a new link**, then **Copy**.

You'll get something like `https://discord.gg/AbCdEfG`.

## 4. Wire the real URLs into the site (1 min)

Open `~/comicteach-site/lib/site.ts` and replace the placeholders:

```ts
links: {
  github:  "https://github.com/comicteach/comicteach",        // ← already correct
  discord: "https://discord.gg/AbCdEfG",                       // ← paste your invite
  twitter: "https://twitter.com/comicteach",                   // ← or remove if you don't make one
  email:   "hello@comicteach.com"                              // ← or your NYU email
}
```

Then re-run dev to verify the buttons work:

```bash
cd ~/comicteach-site
npm run dev
```

---

## 5. Deploy to Vercel (2 min)

1. Sign in to <https://vercel.com> with GitHub.
2. Click **Add New → Project**.
3. Pick `comicteach/comicteach-site`.
4. Framework preset: **Next.js** (auto-detected).
5. Click **Deploy**.

You'll get a `comicteach-site.vercel.app` URL within ~60 seconds.

To wire a custom domain (e.g. `comicteach.com`):

1. Buy the domain (Namecheap, Cloudflare Registrar, etc. — ~$12/year).
2. In Vercel: project → **Settings → Domains → Add** → enter `comicteach.com`.
3. Vercel shows you the DNS records to add at your registrar.
4. Wait ~10 min for DNS to propagate.
